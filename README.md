一个可用的`JSONKit`版本，曾经试过想用系统的`NSJSONSerialization`来替代`JSONKit`。但是当网络数据没有正常获取到的时候，使用`NSJSONSerialization`会导致APP崩溃，所以还是使用`JSONkit`。而原作者的`JSONKit`导入到项目中会报错，这是一个可用的版本。
